# Tamagotchi Challenge

A virtual pet simulation game created with Angular version 9.1.0.

## Usage

### Development server

```bash
ng serve
```

### Build

```bash
ng build
```

### Running unit tests

```bash
ng test
```

### Running end-to-end tests

```bash
ng e2e
```
